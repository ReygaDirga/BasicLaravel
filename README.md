A Laravel-based learning project developed for a **Web Programming course**, covering backend fundamentals from routing to authentication and full CRUD implementation.

---

## Course Overview

| Session | Topic |
|--------|------|
| Session 1 | Introduction to Laravel |
| Session 2 | Routing & Views |
| Session 3 | Laravel UI & Bootstrap |
| Session 4 | Blade Template Engine |
| Session 5 | Controllers |
| Session 6 | Migration & Eloquent ORM |
| Session 7 | Migration & Eloquent (Advanced) |
| Session 8 | Form Processing, Validation, CRUD |
| Session 9 | Collections |
| Session 10 | Localization |
| Session 11 | Session & Middleware |
| Session 12 | Authentication |
| Session 13 | Review |
| Session 14 | Final Project (Collection Integration) |

---

## Key Features

- Implementation of Laravel MVC architecture  
- CRUD system (Create, Read, Update, Delete)  
- Blade templating engine  
- Database management using Eloquent ORM  
- Form validation & request handling  
- Authentication system  
- Middleware & session handling  
- Localization support  
- Data manipulation using Collections  

---

## Tech Stack

- **Backend:** PHP (Laravel)  
- **Frontend:** Bootstrap  
- **Database:** MySQL  
- **Templating:** Blade Engine  

---

## Learning Objectives

- Understand Laravel fundamentals
- Practice backend development concepts
- Implement real-world web application features
- Prepare for full-stack development projects

## Installation

  ```bash
  git clone https://github.com/your-username/BasicLaravel.git
  cd BasicLaravel
  composer install
  cp .env.example .env
  php artisan key:generate
  php artisan migrate
  php artisan serve
